# ipa-archive
A iOS App Dumps (IPA) archive. Nothing special. None is owned by me. All files are tested. Please report outdated files. 

# Types of apps

- Beta versions of apps
- Apps banned from the App Store
- App files in a unstable server
- Modified Apps (Tweaked, Hacked)
- Archived Xcode Project
- Region limited / TestFlight apps
- Hard-to-find Apps
